🐇 Blanc's Recipe (ブランのレシピ)
岩手の霧深い森の中に、霧の朝にだけオープンする不思議なお菓子屋さん。
店主の白うさぎ「ブラン」と、食いしん坊な「あなぐまくん」が織りなす、物語を感じるECサイト風のポートフォリオ作品です。

✨ プロジェクト概要
本プロジェクトは、Webデザインとフロントエンド開発の学習成果として制作しました。
単なるショップサイトではなく、**「ストーリーテリング」と「インタラクティブな体験」**を融合させることをテーマにしています。

🛠️ 使用技術
HTML5 / CSS3 (カスタムアニメーション)

Tailwind CSS (レスポンシブデザイン・UI構築)

JavaScript (Vanilla JS) (動的コンテンツ制御・カート機能)

Lucide Expressive Icons (ベクターアイコン)

Google Fonts (Pacifico / M PLUS Rounded 1c)

🌟 こだわりポイント
1. 「物語」を感じるギミック
あなぐまくんの出現: JavaScriptでメッセージ内容を判別し、「あなぐまくん」という単語が含まれる時だけ画面右下にキャラクターが登場する遊び心を加えています。

店主のメッセージ: ページを読み込むたび、またはあなぐまくんをクリックするたびに、店主ブランからの「今日のひとこと」がランダムに変わります。

2. 世界観を壊さないUI/UX
配色とタイポグラフィ: パステルピンク（#FF8DA1）とミントグリーン（#5F9EA0）を基調とし、角を丸めたデザインで「柔らかさ」と「清潔感」を両立させました。

マイクロインタラクション: スクロールを促すガイドアニメーションや、商品カードのホバー時にアイコンが弾む動きなど、ユーザーが触れていて楽しいと感じる演出を施しています。

3. 実用的な機能の実装
動的な商品レンダリング: JSのオブジェクト形式で管理されたデータから、自動で商品一覧を生成します。

ショッピングカート: ローカルでの簡易的なカート追加・削除・合計金額計算機能を搭載。注文確定時には、あなぐまくんが喜んでジャンプする演出を追加しました。

📱 レスポンシブ対応
モバイルファーストの視点から、スマートフォン・タブレット・PCそれぞれで最適なレイアウトになるよう調整。

画面幅の狭いデバイスでは、操作性を優先して特定の装飾（あなぐまくんのアイコンなど）を非表示にするなどの最適化を行っています。
